# Phishing Incident Report

## Incident ID
PHISH-001

## Incident Type
Phishing Email Attack

## Severity
Medium

## Date
Simulated Incident

## Description
A phishing email impersonating a banking institution was reported by a user.
The email attempted to steal user credentials through a malicious link.

## Indicators of Compromise (IOCs)
- Malicious URL: http://hdfc-verification-secure-login[.]xyz
- Fake sender domain

## Investigation Summary
The email and URL were analyzed using VirusTotal, URLScan, and WHOIS tools.
The domain was newly registered and flagged by multiple security engines.

## Response Actions Taken
- Malicious URL identified and documented
- Incident reported and categorized
- User advised not to click the link
- Awareness recommended

## Final Status
Incident Closed
