# Phishing Investigation Notes

## Alert Source
User reported suspicious email to SOC team.

## Initial Observation
Email contained urgent language and a suspicious external link.

## Investigation Steps
1. Verified sender email domain
2. Analyzed embedded URL using VirusTotal
3. Checked domain age via WHOIS
4. Correlated findings with phishing indicators

## Findings
- Sender domain was not legitimate
- URL flagged as malicious
- Domain recently registered

## Conclusion
The email was confirmed as a phishing attempt based on analysis results.
